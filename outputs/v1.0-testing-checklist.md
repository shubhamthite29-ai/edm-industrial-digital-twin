# EDM Industrial Digital Twin v1.0 Testing Checklist

## Startup

1. Run `Start_EDM_DigitalTwin.bat`.
2. Confirm the gateway reports running on port `8080`.
3. Confirm the dashboard reports running on port `5173`.
4. Confirm the browser opens `http://127.0.0.1:5173/`.
5. Open Unity and press Play.
6. Switch the dashboard to Live Unity Mode.

Expected result: dashboard shows Connected, Unity sends telemetry, Developer Diagnostics packet counters increase.

## Commands

1. Press Start.
2. Press Pause.
3. Press Resume.
4. Press Stop.
5. Press Home.
6. Press Reset.
7. Press Emergency Stop.

Expected result: commands execute without focusing the Unity window, state changes appear in both Unity and React, and the event log records every command.

## Continuous Machining

1. Press Start.
2. Wait at least 60 seconds.
3. Confirm machining continues until Stop or Emergency Stop.

Expected result: progress cycles continuously, elapsed time increases, remaining time updates per cycle, spark state remains active while machining.

## Parameter Synchronization

1. Change Current.
2. Change Voltage.
3. Change Gap Voltage.
4. Change Pulse ON.
5. Change Pulse OFF.
6. Click Preview Apply, then Apply.

Expected result: React sends `machine.parameters.patch`, Unity updates `MachineParameters`, telemetry returns the same values, and settings persist after refresh.

## Telemetry Synchronization

Confirm dashboard displays Unity values for:

- Current
- Voltage
- Gap Voltage
- Temperature
- MRR
- Tool Wear
- Spark State
- Tool Position
- Tank Position
- Machine State
- Elapsed Time
- Remaining Time
- Progress

Expected result: Live Unity Mode uses Unity telemetry as the source of truth.

## Camera Controls

Press:

- Front
- Side
- Top
- Isometric

Expected result: Unity camera changes to the configured transforms.

## Reconnect

1. Stop the gateway.
2. Confirm dashboard shows Disconnected.
3. Restart the gateway.
4. Confirm dashboard reconnects automatically.
5. Confirm Unity reconnects automatically.
6. Press Request Status.

Expected result: state and telemetry recover without refreshing the dashboard.

## Alarms, Events, Notifications

1. Trigger Emergency Stop.
2. Apply parameter changes.
3. Disconnect/reconnect the gateway.

Expected result: notifications appear, event log records actions, and alarm history is retained.

## Shutdown

Run `Stop_EDM_DigitalTwin.bat`.

Expected result: gateway and dashboard processes stop. Unity can be stopped from the Editor Play button.
